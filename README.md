# Web-based Music player
A single-page-application made for streaming music on the web.
## Features
- Unique profile
- stream audio
- Modern minimal design

# Understanding the workflow
## Data
Gets dynamic data from dropbox.
Each user gets their own file created in the cloud.
At the time of login after verification corresponding file is downloaded for changes.
### creating account
Each user gets a unique ID, and the data after creation is fetched this ID.
## UI
The state is controlled primarily by changing the class of app.
mainly three classes
- dark-theme
- light-theme
- blur-on
- horizontal (for PCs)
- vertical (for smartphones)

## Script
All the tabs are created using JS files
